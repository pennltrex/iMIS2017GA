Content Link

  • Name - Enter a name for the content item. This text appears only during design mode (configuration).
  • Internal Link- Select this option to link to a content record within the iMIS website. Not selected = link to external website.
    • Content Record - (Displays if Internal Link is selected.) Browse to select the Content Record for the link or enter the Content Key, Content Code, or Content Record path. Click Preview to view  the Content Record as it will display on the rendered page.
    • Additional URL Parameters - (Displays if Internal Link is selected.) (optional) Enter additional URL parameters for the Content Record URL.  A bracketed URL parameter will be replaced with the actual value from the current request, if any. If there is no value, an empty string will be used. For example, http://www.google.com/gi?key=[iUniformKey], [iUniformKey] will be replaced with the actual iUniformKey from the current request.
  • URL for Link - (Displays if Internal Link is NOT selected.) The URL of an external website. The URL may include additional URL parameters. A bracketed URL parameter will be replaced with the actual value from the current request, if any. If there is no value, an empty string will be used. For example, http://www.google.com/gi?key=[iUniformKey], [iUniformKey] will be replaced with the actual iUniformKey from the current request.
  • Open link in new window - Select this option to open the link destination page in a new window.
