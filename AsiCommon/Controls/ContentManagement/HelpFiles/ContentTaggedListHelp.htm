Content Tagged List

Inserts dynamically generated list of links to content that has been tagged.

  • Name - Enter a name for the content item. This text appears during design mode (configuration).
  • List Format - Select the tagged list format that will display the content item:
    • BlockTaggedList
    • Mobile
    • Short
    • Simple
    • SimpleWithHtml
  • Title - Enter the text to display above this content item on the rendered page. If blank, no title displays.
  • Show title even if no content matches - Enable this option to display the title even if this content item does not match any content.
  • Filter by - Select a data field on which to filter the list results.
    • Relative dates - The current date being when the iPart is viewed will filter the list of results before or after the defined amount of days.
    • Actual dates - Define a specified beginning and ending date to filter the list.
  • Sort by - Select a data field on which to sort the list results.
  • Sort order -
  • Max results to display - Enter the total number of records to be displayed.
  • View all message - Enter a line of text to be used to expand and display all messages.
  • CSS class for view all message - (optional) Enter the name of a CSS class to associate with the View all message configuration. Add this CSS class to an appropriate style sheet. Using such classes lets you apply special formatting to targeted iParts in a reusable way.
  • Include social sharing icons - (optional) Select this option to share content through social media websites.
  • Selected Tags - Click Modify to select tags to be associated with the posts displayed by this iPart. These tags can be used as a search filter. If one or more tags are selected, each post will be associated with each tag. If no tags are selected, all posts belonging to the parent display in search results.
    • Match any tag - Enable this option to have each post be associated with either selected tags.
    • Match all tags - Enable this option to have each post be associated with all selected tags.
