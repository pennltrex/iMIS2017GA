Recurring Donation Commitments Public View (UK)

  • Name - Enter a name for the content item. This text appears during design mode (configuration).
  • Title - Enter the text to display above this iPart on the rendered page. If blank, no title displays.
  • Donation Frequencies - Specifies the donation frequencies displayed in the Per dropdown list.
  • Default Donation Frequency - Specifies which Donation Frequency is selected by default.
  • Payment Method - Specifies the organization's standing order payment method.
  • Distribution - Specifies with which distribution the standing orders are associated.
  • Thank You content to be displayed - Specifies content to be displayed after a user submits a standing order. You can specify content in three ways: click "select" to browse for a content record; enter a relative URL, such as "~/ThisWebsiteFolder/Default.aspx"; or enter an absolute URL. A default thank you page is provided. You can customize it or create your own.
  • Save Button Text - Specifies the text to be displayed on the donation's submit button.

Tip

  • Only authenticated users can make a standing order commitment, so you may want to hide the iPart from unauthenticated users. If so, go to Content Management > Site Builder > Manage sitemaps. Select the navigation item you created for the iPart and click Edit. Expand the Access Settings panel. From the Use a preconfigured security set dropdown list, select an option to only allow access to authenticated users, such as Authenticated Users Full Control. Save your changes and then Publish the navigation item. The navigation item no longer appears for unauthenticated users.
