Gift Aid Declaration

  • Name - Enter a name for the content item. This text appears during design mode (configuration).
  • Title - Enter the text to display above this iPart on the rendered page. If blank, no title displays.
  • Default behaviour for Effective From date - To boost performance during design time (configuration), enable this checkbox to prevent content record data from being loaded in the content editor.
    • Apply the 4-year rule - Sets the effective date of the declaration to a rolling four-year period that equals today minus 4 years.
    • User will enter an 'effective from' date - Sets the effective date of the declaration to a fixed date from which the declaration is valid.
  • Default GAD duration (in years) - Specifies the default length of the declaration in years, which sets the default Expiry Date.
  • Default declaration method - Specifies which Declaration Method option you want selected by default.
