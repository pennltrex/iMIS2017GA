Item Creator

The Item Creator iPart allows new items to be added to the Shop list. It also creates diverse products for sale, such as mechandise, billing and subscription items, and publications. In addition, you can create fundraising donation items (gift items) using this content item.

  • Name - Enter a name for the content item. This text only appears during design mode (configuration).
  • Title - Enter the text to display above this iPart on the rendered page. If blank, no title displays.
  • Do not render in design mode - Enable this option to prevent content record data from being loaded in the content editor. This improves performance during design time (configuration).
  • Display a border around this content - Enable this option to display a border around this iPart on the rendered page.
  • Item type - Select the item type that will be created:
    • Both - Selecting Both will enable you to create either a Product or Gift item using this content item.
    • Product - Selecting Product will allow you to only create a commerce product using this content item. Out-of-the-box from Commerce > Add product, Product is selected by default.
    • Gift item - Selecting Gift will allow you to only create a donation gift item using this content item. Out-of-the-box from Fundraising > Add gift item, Gift item is selected by default.

Notes

  • When a user creates a new item, if the Item Class = Gift Item, the Suggested Amount option replaces the Standard Price option.
  • This iPart supports the passing of the following URL parameters:
    • iProductID: Item ID of the item to edit. (a GUID)
    • iProductCode: Item Code of the item to edit
  • If no parameters are passed, the iPart will be displayed in "Add" mode. If both parameters are passed, only the iProductID will be honored. If either parameter is passed and the matching product exists, the iPart will be displayed in "Edit" mode. Otherwise, it defaults to "Add" mode.
