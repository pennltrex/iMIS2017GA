Secondary Navigation

This content item is used to display the secondary navigation of the website. This is usually displayed on the left or right side of the page.
Note: The secondary navigation content will display only if the selected page has secondary navigation content associated with it.
  • Name - Enter a name for the content item. This text appears during design mode (configuration).
  • iPart CSS class - (optional) Enter the name of a CSS class to associate with this iPart, which will add a DIV so named around the iPart. Add this CSS class to an appropriate style sheet. Using such classes lets you apply special formatting to targeted iParts in a reusable way.
    • Skin - Enter the name of the custom skin that will be applied to the secondary navigation.
    • Starting at - The selected level introduces the secondary navigation. If a selected page does not contain this level of navigation, the secondary navigation will not display.
    • Display - After the Starting at level is defined, select the depth of levels to be displayed in the secondary navigation.
